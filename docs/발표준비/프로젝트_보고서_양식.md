## 부록 A. 서식 규격 (양식 docx에서 추출한 실제 값)

docx로 옮길 때 이 값과 다르면 안 된다. 양식 스타일을 그대로 쓰면 자동으로 맞는다.

| 요소 | 글꼴(한글/영문) | 크기 | 정렬·기타 |
|---|---|---|---|
| 기본 본문 | 바탕체 / Times New Roman | 10pt | 양쪽 정렬, 첫 줄 들여쓰기 |
| 국문 제목 | 바탕체 | 16pt 굵게 | 가운데 |
| 영문 제목 | 바탕체 | 16pt 굵게 | 가운데 |
| 저자명 | 바탕체 | 10pt | 가운데 |
| 이메일 | 바탕체 | 9pt | 가운데 |
| 요약 | 바탕체 | 9pt | 좌우 여백 들여쓰기 |
| 장 제목(1, 2, 3) | 돋움체 | 굵게, 줄간격 180% | 번호 자동 |
| 그림·표 캡션 | 바탕체 | 9pt | 가운데 |
| 참고문헌 항목 | 바탕체 | 10pt | 들여쓰기 없음 |

- 용지: A4 세로 / 여백 위 30mm, 아래 17mm, 좌우 15mm 내외 / 단 구성: 1단
- 캡션 위치: **그림은 아래**, **표는 위**. 표기는 `(그림 N)`, `<표 N>` 형식 그대로.

## 부록 B. 분량 배분 (최소 4p / 최대 6p)

| 구간 | 목표 분량 | 그림·표 |
|---|---|---|
| 표지 + 요약 | 0.5p | – |
| 1. 프로젝트 개요 | 0.7~1p | – |
| 2. 프로젝트 내용 | 2~2.5p | 그림 1, 표 1 |
| 3. 프로젝트 결과 | 1.5p | 표 2, 그림 2~3 |
| 참고문헌 | 0.3p | – |

5페이지를 목표로 쓰면 편집 후 4~6p 안에 안전하게 들어온다.

## 부록 C. 제출 전 체크리스트
