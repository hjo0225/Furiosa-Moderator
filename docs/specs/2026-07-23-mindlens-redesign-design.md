# mindlens 홀리스틱 리디자인 — 설계 스펙

> 하네스 Brainstorm 산출물(§3). 대상 = mindlens 3개 표면(의뢰자 대시보드·응답자 인터뷰·랜딩) 전면 리디자인.
> 시각 방향 보드(아티팩트): 대화 중 발행. 디자인 시스템 정본 = `design.md`(이 스펙과 함께 갱신).
> 상태: **설계 확정 대기 → 사용자 리뷰 → Plan.**

## 1. 목적 · 성공 기준
- **문제:** 현행 UI가 "데모" 같다 — 앱 셸 부재(화면마다 `← mindlens` 텍스트 링크), 이모지 아이콘, 플레이스홀더 상태("불러오는 중…"), 얇은 랜딩, 낮은 데이터 밀도, 디자인 문서 드리프트(stale Boram design.md).
- **북극성:** **Furiosa 부트캠프 쇼케이스** — 라이브 워크스루에서 프로덕션 제품처럼 보이고, 하드웨어 스토리(NPU)가 자연스럽게 실린다. 일관성 + 소수의 임팩트 화면 > 엣지케이스 전수.
- **성공:** ① 3개 표면이 하나의 코히런트 시스템 ② 데모 티(이모지·플레이스홀더 상태·셸 부재) 제거 ③ 벤치마크가 하드웨어 우위를 정직하게 전달.

## 2. 확정 결정
1. **시각 목소리:** 따뜻한 리서치 툴(Typeform/Maze) 소울 + **Furiosa 브랜드 색 채택**. 최종 = **이원화**(도구=정통 / 대화=웜), 그라운드는 라이트(블랙 드라마 배제). → `design.md §0`.
2. **앱 셸(의뢰자):** 좌측 사이드바(A1). **무인증** 반영(계정/워크스페이스 없음, "링크만 공유" 안내). → `design.md §5`.
3. **색:** furiosa.ai 실측 토큰. 액센트 = `brand-red #E21500`. 중립 obsidian/silver. **에러는 maroon으로 분리**(brand-red 충돌 회피). 데이터=바이브런트 팔레트. → `design.md §1`.
4. **아이콘:** **`lucide-react`, 이모지 금지**(하네스 §2). 매핑표 = `design.md §4`.
5. **타이포:** Pretendard(크고 얇은 400 타이트) + `tabular-nums`. **벤치마크 텔레메트리에만 시스템 mono.** → `design.md §2`.
6. **상태 화면:** 스켈레톤/빈(아이콘+CTA)/에러(maroon+재시도). "불러오는 중…" 텍스트 폐기.
7. **인터뷰(응답자):** **현행 구조 그대로 + 웜 스킨.** 진행자 = 작은 웜 오브(음성 반응 맥동). 블랙 아님.
8. **벤치마크(의뢰자):** RNGD 계측 스펙 §7 소비 — 손익분기 S\* + M1/M2/M3. 미측정=null, 추정치 금지. NPU=red/GPU=grey. 측정 하네스는 별도 워크스트림.
9. **랜딩:** 현행 구조 + 라이트 스킨(블랙 없이).

## 3. 범위
- **In:** web/ 전 화면 리스킨(셸·대시보드·가이드·결과·인터뷰·동의·스크리너·완료·랜딩) + 신규 벤치마크 대시보드 뷰 + 상태 컴포넌트 + 아이콘 이모지→lucide 횡단 교체 + `design.md`/`globals.css @theme`/`tailwind.config.ts` 토큰 정합.
- **Out(별도 워크스트림):** 벤치마크 **측정 하네스**(부하 재생기·PDU/Prometheus 수집·κ 골드셋 — `docs/specs/2026-07-23-rngd-benchmark-instrumentation.md`). 백엔드/모델 변경. 인증 도입.

## 4. 컴포넌트 인벤토리 (구현 단위)
- **시스템:** 토큰(globals.css @theme·tailwind), 타이포 유틸(tabular-nums·mono 스코프), 아이콘 셋 배선(lucide-react).
- **셸:** `Sidebar`(무인증), 콘텐츠 헤더바.
- **공통:** `Button`(primary/secondary/ghost), `Card`, `Badge`, `EmptyState`, `ErrorState`, `Skeleton`.
- **의뢰자:** 프로젝트 리스트 카드, 가이드 패널, 결과 대시보드(차트 리컬러), **`BenchmarkView`(신규)**.
- **응답자:** `InterviewFlow`(리스킨, 구조 불변) + `ModeratorOrb`(신규, 이모지 대체), 동의·스크리너·완료.
- **랜딩:** 히어로·스텝·응답자 프리뷰·푸터.

## 5. 하네스 게이트 · 리스크 (Plan에서 못박을 것)
- **횡단 전수조사(§2):** 이모지→lucide 교체 = 횡단 → 모든 소비처 grep(컴포넌트·테스트·aria). 토큰 리네임(`--accent*`→red) = 횡단 → globals.css·tailwind·차트 리터럴·컴포넌트 클래스 전수.
- **계약 보존:** 리스킨은 UI만 — 집계=DB 실측, PII 마스킹 저장경로, transcribe (전사,성공) 계약(§0.1)을 **건드리지 않음**.
- **하드게이트(§1):** globals.css/컴포넌트 편집은 Implement. 커밋/PR 초안까지 자율, **머지=배포는 사람 승인.**
- **검증:** `tsc --noEmit`·`next build`·`pytest`(파이썬 무변경 예상)·**UI 스크린샷**·`/design-review`(design.md 토큰 준수).
